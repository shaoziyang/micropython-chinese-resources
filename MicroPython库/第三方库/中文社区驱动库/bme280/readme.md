# BME280 温湿度气压传感器

BME280 是博世（Bosch Sensortec）推出的三合一数字式环境传感器，能够同时测量 温度、相对湿度和大气压力，并在此基础上计算海拔高度。它体积小、功耗低、精度高，常用于天气站、物联网、可穿戴设备、无人机导航等场景。BME280 是 BMP280 的升级，也是目前最常用的型号之一。

```
      ┌─────────────┐
  VDD │ 8         1 │ VSS
  SDA │ 7         2 │ SCK
  SDO │ 6         3 │ CSB
  NC  │ 5    O    4 │ SDI
      └─────────────┘

```

## 主要特性
- **封装规格**：2.5毫米×2.5毫米×0.93毫米金属盖网格阵列封装（LGA）
- **数字接口**：支持I²C（最高3.4 MHz）和SPI（3线与4线模式，最高10 MHz）
- **供电电压**：
  - 主供电电压（VDD）范围：1.71伏~3.6伏
  - 接口供电电压（VDDIO）范围：1.2伏~3.6伏
- **电流消耗**：
  - 湿度与温度检测（1 Hz采样率）：1.8微安
  - 气压与温度检测（1 Hz采样率）：2.8微安
  - 湿度、气压与温度同时检测（1 Hz采样率）：3.6微安
  - 休眠模式：0.1微安
- **工作范围**：
  - 温度：-40℃~+85℃
  - 相对湿度：0%~100%
  - 气压：300百帕~1100百帕
- **传感器独立控制**：湿度传感器与气压传感器可独立启用/禁用
- **兼容性**：寄存器与性能均兼容博世传感器技术公司（Bosch Sensortec）的BMP280数字气压传感器
- **合规性**：符合RoHS标准，不含卤素，符合MSL1（ moisture sensitivity level 1，湿敏等级1）要求


## 湿度传感器关键参数
1. **响应时间（τ63%，即达到63%阶跃响应的时间）**：1秒
2. **精度误差**：±3%相对湿度
3. **滞后误差**：±1%相对湿度


## 气压传感器关键参数
- **均方根噪声（RMS Noise）**：0.2帕，相当于1.7厘米（水位高度差）
- **偏移温度系数**：±1.5帕/开尔文（K），相当于温度变化1℃时，误差±12.6厘米（水位高度差）

## **使用方法** 
需要先将 [bmp280 驱动](https://gitee.com/microbit/mpy-lib/tree/master/sensor/bme280) 复制到开发板中：

```py
from machine import I2C
import time

import bme280

b = bme280.BME280(I2C(1))

while True:
    time.sleep_ms(500)
    print(b.get())
```
