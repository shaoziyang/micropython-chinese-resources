# LIS2DH12TR 加速度传感器

LIS2DH12TR 是 ST 超低功耗、高性能的三轴线性加速度计，属于“femto”系列，具有数字I2C/SPI串行接口标准输出。LIS2DH12具有用户可选择的量程，包括±2g、±4g、±8g和±16g，能够测量加速度，输出数据速率范围为1 Hz至5.3 kHz。
