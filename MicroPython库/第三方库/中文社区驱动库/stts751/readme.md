# STTS751 数字温度传感器

STTS751 是 ST 的数字温度传感器，通过 SMBus 总线进行通信。温度以9到12位的用户可配置分辨率进行测量。在9位时,最小步幅大小为0.5°C,12位为0.0625°C。在默认分辨率下(10位,0.25°C/LSB)，转换时间为21毫秒。
