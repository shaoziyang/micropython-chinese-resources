BasicPython 是一款用于 CircuitPython 设备的交互式 Python shell，设计为完全无需计算机即可工作。它最初受到斯科特·肖克罗夫特basicpython的启发，并为beboxos的Adafruit PyPortal Titano进行了移植/扩展。
可以用于：
- 按行编写和运行Python程序(基本样式)
- 管理设备闪存存储上的文件和目录
- 直接计算Python表达式
- 检查内存和磁盘使用情况
- 只需要一个微型 I2C 键盘，无需USB线缆。

项目网址：
[https://github.com/beboxos/circuitpython/tree/main/pyportal titano/basicpython](https://github.com/beboxos/circuitpython/tree/main/pyportal%20titano/basicpython)
