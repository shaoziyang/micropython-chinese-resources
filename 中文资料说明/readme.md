# 中文资料说明

社区在2016年时曾经发布过旧版本的【 micropython中文教程】，当时micropython才正式发布2年，版本刚到 v1.8；而现在的 micropython 已经更新到了 v1.27 版，功能上已经有了很大变化，增加了很多新的功能，甚至有些用法也不完全相同，支持的硬件更多，功能也更强，API更加规范了。今年也是 micropython 开发的第12年（micropython是从2013年4月29日写下第一行代码的），旧的中文教程已经不太适合现在的 micropython，因此需要对内容进行大量更新。考虑到直接对旧版本进行改动会比较复制，因此创建了这个新的项目：【micropython 中文资料】，这个项目将搜集、整理、展示最新的 micropython 中文技术资料，方便大家参考。

新版本的 micropython 中文资料将不断搜集整理 micropython 的相关教程、资源、开发技巧、应用，并持续跟进 micropython 的最新变化，及时将这些内容同步发布。新的 micropython 中文资料已经同步发布到 gitee、 github、 飞书文档，它们的内容基本是一致的，但是因为gitee在国内的访问速度更快，所以内容通常优先在 gitee 上更新。
