# micro:bit 群模拟器

使用 MicroPython 和 GitHub 模拟 Micro:bit 群。

群模拟器是一个画布，可以创建micro:bit节点以及某些类型的环境节点(如光和声源)，然后将预写的设备代码加载到设备上，查看它们的效果。可以拖动不同的节点，查看变化，按下按钮、检查无线电，更好地了解在真实环境中真实设备会发生什么。

**它支持:**

- 显示
- 按钮
- 无线电（范围、分组和RSSI 值，通道和传输电源还需要测试，可能无法运行）
- 声音发射的视觉效果
- 串口输出
- 数据日志（通过Swarm工具下载一个压缩文件包）
- 光线与声音传感器
- 基于测试和原型设计的源语言，对 main.py 和 main.ts 进行轻量级代码编辑（仅文本，需要先加载代码）
- 使用浏览器（IDB）或外部文件中的设备代码和位置保存并加载画布设置。将这些内容与您自己的项目一起使用，或与学生分享
- 锁定没有进行代码编辑的 micro:bit，只支持单个程序加载。利用这些内容来探究学生需要探究的奥秘。
- 锁定的 micro:bit 也可以固定。
- 可以配置画布指令，并在加载画布状态、保存的浏览器状态或上传的文件时显示。

**不支持**

-	在这里不会创建任何代码。在其它程序中编写然后拖放进去。
-	这不是物理模拟器，而是一个“合理的近似模拟器”
-	这并非精确的 micro:bit 模拟器。故意省略了一些东西（如加速度计、引脚）
-	无线电交互比现实世界中更为具有兼容性。
