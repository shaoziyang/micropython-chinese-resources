# MicroPython入选谷歌文档季

Google推出的Season of Docs (SoD) 项目旨在提升开源项目文档质量，SoD每年组织技术作家参与为期三个月的文档工作，2020年MicroPython入选该计划，MicroPython的参与旨在借助专业技术作家优化其文档体系。
