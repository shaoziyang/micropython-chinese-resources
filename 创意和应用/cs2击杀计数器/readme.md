# 反恐精英 2 击杀计数器

一个《反恐精英 2》玩家制作了一个实体击杀计数器，它放在桌子上，并实时更新，为玩家在游戏中提供即时反馈。设备使用树莓派 Pico 搭配 Pimoroni Galactic Unicorn LED 阵列制作，直接放在显示器下面。

![](https://cdn-blog.adafruit.com/uploads/2026/01/20260119score.gif)

让这个项目脱颖而出的关键在于数据的收集方式。创建者并没有修改游戏或以任何方式读取内存。相反，该系统使用的是 Valve 的游戏状态集成（GSI），这是一个官方功能，允许游戏外部共享比赛数据。这些数据被发送到一个 Python 脚本，该脚本随后通过 USB 与 LED 矩阵通信以立即更新显示。这使设置保持整洁，并避免了任何作弊问题。

计分器会在游戏中登记击杀时立即更新。除此之外，创作者还添加了与不同事件相关的视觉效果。当玩家获得 MVP 时，显示屏会触发金色烟花式动画。当地图加载时，整个面板会闪烁明亮的白色，模拟闪光弹的效果。在休息室中，显示屏会循环播放 AK-47 喷漆图案的动画，即使在不进行比赛时也能保持屏幕活跃。

该设备完全作为外部配件运行，不会干扰游戏。它只是对游戏数据进行反应，并以有趣的方式将其可视化。对于有兴趣自己动手制作的玩家，创作者已将整个项目开源。代码、硬件清单和设置说明可在 GitHub 上找到，使其他人可以轻松地重现或扩展这个想法。

**使用的硬件**
- Pimoroni Galactic Unicorn （RP2040 LED 矩阵）
- MicroUSB 数据线
- 运行《反恐精英 2》的电脑
