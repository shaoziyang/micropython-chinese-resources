# LAMPy

LAMPy 是一款基于 MicroPython 的 RGB 灯控制器。它首先是一个软件项目，意味着它可以运行在许多不同的硬件上。唯一的要求是带有天线的ESP32板、一个WS2812(NeopPixel)LED灯带以及一个按钮。

![](lampy.webp)

**特点**

- 可调节的亮度，带有视觉指示灯。
- 带有可选颜色的固定颜色模式。
- 具有可调节动画速度的色彩模式。
- 多个灯可以无线同步。
- 过热保护。
