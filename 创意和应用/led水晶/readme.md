# LED 水晶

3D 打印的八面水晶，可以通过 Octoprint API 显示连接的打印机状态，使用 circuitpython 编程。LED 有8种不同的模式:怠速、打印、精加工、加热、制冷、错误、暂停和连接错误，在大多数模式下 LED 以动画方式显示。
