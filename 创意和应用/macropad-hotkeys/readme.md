# 快捷键宏键盘

![](https://cdn-blog.adafruit.com/uploads/2026/05/ezgif-6ae241bdc46f3371.gif)

这是 adafruit learn 中的一个代表项目，通过小的宏键盘发送快捷键。

按下宏键盘的12个按键之一，将快捷键、功能键或整个按键序列发送到已连接的计算机。OLED显示屏显示按键功能，而每个按键下方的LED灯通过颜色显示分组或主题。转动表盘，可以在不同的应用程序集中进行切换。

这是你可以简单找到日常使用的项目之一，也可以在代码中查看CircuitPython如何使这一切变得相当简单。此外不同桌面应用程序的热键配置文件可轻松创建、修改和共享。
