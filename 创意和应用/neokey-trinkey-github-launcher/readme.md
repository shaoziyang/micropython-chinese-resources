# NeoKey Trinkey GitHub Launcher

将 NeoKey Trinkey 开发板变成 GitHub 个人资料的快速启动按钮。只需按下按钮，它会自动打开 Firefox 并导航到你的 GitHub 页面。使用 circuitpython 编程。

![](neokey_trinkey_github_launcher.webp) 

## 特色
- **一键启动 GitHub** - 按下按钮即可立即打开您的 GitHub 个人资料
- **视觉反馈** ——蓝色 LED 表示准备就绪，按下按钮时绿色 LED 闪烁
- **USB HID 键盘** ——使用 HID 键盘模拟与电脑交互
- **自动浏览器启动** ——打开 Firefox 并导航到你的 GitHub URL
