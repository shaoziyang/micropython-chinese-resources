# TinyWiki

使用 MicroPython 创建的 Tiny Wiki。它可以在任何支持网络功能的 MicroPython 设备上运行，非常适合创建家庭维基。

## 特色

* Markdown 支持 ：在 Markdown 中编写和渲染页面
* 维基链接 ：使用 [[PageName]] 语法连接页面
* 网页界面 ：通过浏览器编辑和查看页面
* 自托管 ：所有数据存储在文件系统上
* 轻量级：针对 MicroPython 的资源约束进行了优化
* 无数据库 ：基于文件的简单存储

## 硬件

* Raspberry Pi Pico W (or Pico 2W)
* ESP32-based board with MicroPython
* Arduino Nano RP2040 Connect with MicroPython
* Arduino Nano ESP32 with MicroPython

## 软件

* MicroPython firmware (v1.20 or later)
* microdot web framework
