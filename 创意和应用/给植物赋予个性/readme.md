# 给🪴植物赋予个性

用 Raspberry Pi Pico W 给🪴植物赋予个性。用 MicroPython 在 Raspberry Pi Pico W 上编程，配合 Pimoroni 种植套件和 Twilio，打造出能给你发短信的植物！

![](https://www.raspberrypi.com/app/uploads/2026/02/Texting-Pot-Plant-1024x875.jpg)

可以通过加入额外的信息，赋予植物个性，扩展项目并使其更有趣：

- 早安☀️晚安😴——基于光线传感器和一天中的时间
- 随机🎲信息
- 我需要一些水💧，谢谢你的水🙏——基于土壤湿度传感器
