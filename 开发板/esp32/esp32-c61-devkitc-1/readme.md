# ESP32-C61-DevKitC-1

ESP32-C61-DevKitC-1 是一款入门级开发板，使用通用型模组 ESP32-C61-WROOM-1。该开发板搭载的模组配置了高达 8 MB SPI flash 和 2 MB PSRAM。该款开发板具备完整的 Wi-Fi 和低功耗蓝牙功能。
