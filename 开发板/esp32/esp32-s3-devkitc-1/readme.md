# ESP32-S3-DevKitC-1

乐鑫官方的 ESP32-S3-DevKitC-1 开发板，带有 2 个 USB 接口（一个原生 USB 接口和一个 USB 转串口），两个 USB 接口都可以下载程序。它有几种不同型号，主要区别在于模块内部的 flash 和 psram 的大小不同。
