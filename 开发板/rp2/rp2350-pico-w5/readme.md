# RP2350 Pico W5

![](pico_w5_with_rp2350_chip_1.webp)

Elecrow 推出的 Pico 2 W 升级版本，尺寸和功能上兼容标准的 Pico 2 W，但是支持 5G 无线信号，存储容量增加到 8M，带有复位按钮，以及使用了 usb type c 接口。
