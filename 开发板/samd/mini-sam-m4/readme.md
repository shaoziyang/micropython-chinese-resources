# Mini SAM M4

Mini SAM M4 是一款基于 SAMD51G 的 32 位 ARM® Cortex-M4F® MCU 的人形开发板，运行频率为 120Mhz，具有 18 个 I/O、两个内置 LED、按钮和一个额外的 2MB 内存。

Mini SAM M4 具有多种不同颜色以及变种。
