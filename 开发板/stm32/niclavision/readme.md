# Arduino Nicla Vision

Arduino Nicla Vision 集成了强大的 STM32H747AII6 ARM® Cortex® M7/M4 IC 处理器和支持 TinyML 的 200 万彩色摄像头，以及智能六轴运动传感器、集成麦克风和距离传感器。
