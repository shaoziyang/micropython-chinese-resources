# Visual Studio Code

Visual Studio Code （简称 vscode） 是微软推出的免费、开源、轻量级、可扩展的代码编辑器，它功能强大，配合各种扩展可以实现各种功能。如果习惯使用vscode进行开发，可以使用RT-Thread MicroPython扩展，方便的开发micropython程序。

在vscode的扩展中，搜索MicroPython就可以找到RT-Thread MicroPython扩展，它的特点有：

* 设备快速连接（串口、网络、USB）
* 支持基于 MicroPython 的代码智能补全与语法检查
* 支持 MicroPython REPL 交互环境
* 提供丰富的代码示例与 demo 程序
* 提供工程同步功能
* 支持下载单个文件或文件夹至开发板
* 支持在内存中快速运行代码文件功能
* 支持运行代码片段功能
* 支持多款主流 MicroPython 开发板
* 支持 Windows、Ubuntu、Mac 操作系统
