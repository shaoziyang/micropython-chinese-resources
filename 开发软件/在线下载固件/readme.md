# 在线下载固件

## ESP32

通过浏览器，直接下载固件到esp32芯片，无需安装esptool、flash downloader等软件。

需要使用支持webusb功能的 chrome 内核浏览器，不支持 Safari、Firefox等浏览器。

* https://espressif.github.io/esp-launchpad/
* https://adafruit.github.io/Adafruit_WebSerial_ESPTool/
* https://esp.huhn.me/

## RP2

* picoflash: 使用 WebUSB 直接从您的网页浏览器更新树莓派 Pico 微控制器。
    * [网站](https://picoflash.org/)
    * [github 仓库](https://github.com/piersfinlayson/picoflash)
