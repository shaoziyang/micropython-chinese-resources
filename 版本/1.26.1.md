# v1.26.1，2025-09-11

**ESP32原生USB支持补丁版本**

本补丁版本包含以下提交内容：

- esp32：将 esp_tinyusb 组件更新至 v1.7.6 版本
- 工具：新增环境变量 MICROPY_MAINTAINER_BUILD
- esp32：将 IDF 组件锁定文件（IDF Component Lockfiles）添加至 git 仓库
- shared/tinyusb：修复因新增 tx_overwritable_if_not_connected 标志导致的挂起问题
- shared/tinyusb/mp_usbd_cdc：重写 USB CDC （USB通信设备类）的 TX（发送）循环
- tools/mpremote：不对 TinyUSB CDC 设备应用乐鑫（Espressif）的 DTR/RTS 特性修正
